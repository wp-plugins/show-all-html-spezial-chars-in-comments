msHTMLspecialChars - Show all HTML special chars in your comments. 

== Description ==

With msHTMLspecialChars can you see all HTML special chars in your comments.
Bugreports and more informations find you <a href="http://www.zauberpage.de/show-all-html-spezial-chars-in-comments.html">here</a>.

== Installation ==

1. Download msHTMLspecialChars Plugin 
2. Unzip file
3. Upload the folder "ms-htmlspecialchars-comments" into your /wp-content/plugins directory
4. activate Plug-In
5. Have fun with msHTMLspecialChars :)
